How auto-selection works

System rules are designed for common apparel tech pack structures and can be refined later by buyer / customer profile.

Text-first logic

  • Looks for comment / remarks / revise / amend keywords
  • Prefers pages with enough text content
  • Includes measurement / BOM pages when relevant

Image-markup logic

  • Looks for low-text pages, sketch / artwork / placement clues
  • Boosts pages after measurement tables
  • Useful when buyer marks directly on images

Hybrid logic

  • Uses text pages and image pages together
  • Best default when customer method is mixed
  • Final report merges text and visual findings
